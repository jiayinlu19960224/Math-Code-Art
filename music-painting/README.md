# Music Painting

## Brief Description
This tutorial demonstrates how musical information can be extracted from audio pieces using the librosa library, and how to use mathematical functions to map the input music to RGB colors. Then, these information can be used to generate random walk paths with varying widths and directions, which produces artistic visuals synchronized with the underlying audio.
