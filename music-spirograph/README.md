# Music Spirograph

## Brief Description

Music Spirograph is a music-reactive spirograph renderer built with Python, Matplotlib, and Librosa.  
It uses four moving entities with invisible circular colliders to produce smoother motion and clearer collisions while rendering synchronized geometric patterns to video.

Main files:
- `Spiro_Original.ipynb`
- `Spiro_Stable_From_Original.ipynb`
- `Spiro_Stable_Tutorial.ipynb`
